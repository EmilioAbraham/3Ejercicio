Titulo del proyecto: Ejercicio 3  obtener la diferencia de dos numeros introducidos en la misma linea
Proposito del proyecto: restar dos numeros ingresados en la misma cadena 
Version o fecha: 28/05/2020
Como iniciar este proyecto: Ejecutar el main de la clase
Autores: Emilio Abraham Arreola Cruz 19161240
Instrucciones para el usuario:
Se deberan ingresar dos numeros separados por un espacio
Solucion: 
 Primero se deberan ungresar dos cantidades separadoas por un espacio , despues se lee la cadena  y  se va guardando en una variable tipo String hasta que se lea un espacio en blanco, entonces lo siguiente se guarda en  ortra variable, quese convertiran a tipo entero y  se realiza la resta correspondiente
